---

## 🚀 Why Mess Buddy?
Managing a mess involves complex calculations: daily bazar costs, varying meal counts, and "Minimum Meal" policies. Mess Buddy automates this logic using a **Real-Time Database**, ensuring everyone is on the same page (literally).

### ✨ Key Features
- **Live Sync:** Powered by Firebase; updates on one device reflect instantly on all others.
- **Automated Billing:** Calculates individual dues based on the current meal rate and personal consumption.
- **Admin Tools:** Secure access to log daily expenses, manage the team, and post global announcements.
- **UPI Integration:** Dynamic QR code generation for instant debt settlement.
- **Privacy Mode:** "View My Stats Only" toggle for a focused member experience.

## 🔧 Installation & Setup

### For Web:
1. Clone the repo: `git clone https://github.com/yourusername/mess-buddy.git`
2. Open `index.html` in your browser.
3. Update the `firebaseConfig` object in the script tag with your own credentials.

### For Android:
1. Import the project into **Android Studio**.
2. Add your `google-services.json` to the `/app` folder.
3. Sync Gradle and run on your emulator or device.
